A peaceful walk · 襄阳日记
Oct. 24th, Memory of Xiangyang

古城旧梦

一江碧水穿城过，十里青山半入城。襄阳，这座被宽阔的汉江一分为二的城市，带着一种奇妙的撕裂感。

它的一面，是金庸笔下死守襄阳的侠骨悲壮，是历史书里铁打的襄阳的冷硬；而另一面，我们在襄阳城里切实的感受又截然不同，襄阳是个安静的城市。鼓楼门前的大道车水马龙却不显得繁杂，襄王府的街口行人匆匆又格外宁静。
Continued...

幸福

我们在汉江边漫步，不见人们太多步履匆忙，孔明灯高飞着，模仿古时晴朗夜空的星，我们在古城墙上相拥着，期待这份宁静久远一点、悠长一点。

古时的喧闹与你我无关，夜风拂过古老的垛口，吹散了金戈铁马的幻梦。我想问，古代的人们也吃襄阳红油牛肉面吗？战乱的年代，肯定有许多如同你我一样真挚的感情，却在烽火连天中消散。我们依偎在岁月的一角，这样的日子于我而言，已经足够幸福.

May. 22nd, Memory of Chengdu

萧

暴雨的武汉，安详的成都，我们度过的第一个春季，悄悄在街头巷尾离去。乐山，回来后总想起，那个文化交融的年代，武则天信奉“弥勒出世，天下太平”，佛庙林立。

这样的主题，我们又在敦煌有所见证，粉饰的太平，守不住万里江山；石佛无言，雨水和变革后，只剩王权幻梦、只剩崖壁残痕。
May. 22nd, Memory of Chengdu

泰山还是鸿毛

可是人生百变，不止历史无涯，和你在一起，反而常觉得历史虚妄。

乐山有什么，长什么样，连同乐山一块，落在四川了，我们彼此相望的时候，就应该明白，一辈子能够记住的，有多少呢。
May. 22nd, Memory of Chengdu

渺小也可以伟大

我们见了一只举止优雅，佛性初现的三花。我们去过的许多与佛有关的地方，似乎都有许多小猫，与破败的庙堂，憔悴的佛像相对应的是生机的蓬勃。

佛说：一花一世界，一叶一菩提。真理似乎不在宏大的崇拜之中，而是对一切渺小的尊敬，是虚妄人生中的真情。在时代面前，我们比高大的佛像更脆弱，我们的爱却比温暖的生灵更炽热。

Dec.21th, Memory of Jingde

泥土之城

景德镇它的底色是泥土的厚重，灵魂是烈火的狂热，呈现给世人的，却是如水般的温润与清透。我们漫步在陶溪川，这个已成为过去式的工业瓷厂，如今在兜售梦想，这里有，保留着泥土粗粝与火焰吻痕的拙朴柴烧，色彩明快、画满童话碎片的家庭手绘；打破常规、泛着金属光泽的先锋解构艺术；还有你带着头上的小黄帽，都在诉说这里的美丽。
Dec.21th, Memory of Jingde

无用之用

美丽是种表象，有些人的人生目的在这里，在陶溪川过客匆匆的摊子上，在陶阳里大大小小的作坊中，我们慌忙乎急略过的小物件，可能是一个人的全部归属。

景德镇允许“无用之用”，这里没有严苛的世俗成功学标准，只有对泥土的专注，我喜欢景德镇，可能是因为我喜欢这份热闹的恬静，我们骑着小电驴满城的跑，风把我吹成傻瓜蛋，但是也把生活的疲惫远远的甩在后头，
Dec.21th, Memory of Jingde

生活是什么

我们吃着景德镇的江西菜，喝着joketea!在御窑博物馆出片，懒病犯了就在床上多躺一会。

我喜欢和你一起能够操纵生活的日子，而不是总是被一堆事情推着滚滚向前。

River and Mountains · 三峡日记
Oct. 18th, Memory of Sanxia

万物自有其时

时间，就是你珍视的成了过往，你的生活成了曾经，时间是西陵峡迎着游船吹来的风，三峡大瀑布蒸腾的雾气，伸出手来抓不住，多么短暂。可这正是我们为什么爱，我们为什么欢笑，我们为什么感动，因为这无法挽留的流逝，赋予了每一个当下不可复制的重量。那节列车人很少，嬉戏打闹着，车就到站了，站外，是一连串整齐的出租车，我们到了酒店，酒店给我们升了个套房，我们在楼下老火锅吃饭，被辣的不轻，
Continued...

万物自有其时

晚上你在洗澡，我只能去楼下上厕所，结果把自己锁在门外，三峡大瀑布的两个老鼠被互换元宝都一年多了，三峡人家的那艘游船不知又迎送了多少匆匆的过客，别人的记忆或许早随江水远去，更无从知晓，在这艘再平凡不过的江船上，曾航行过两个年轻人不可替代的相爱岁月，时间尽可以残酷的把一切变成过往，但在我这里，你是我最珍视的当下。

龟山纪行

龟山很安静，比龟山更安静的，是有一个吵闹名字的叽里呱啦

Nov. 12th, Memory of Guishan
武昌江滩堆满游人，蛇山车水马龙，黄鹤楼灯火通明，璀璨如白昼，可吵闹永远过不了这道江，武汉的江风总吹的过急，激狂的白浪把喧嚣击的粉碎，长江与汉水在南岸嘴劈波相撞的那一刻，光阴在此处落下了江城静默的留白。所以，我们登上龟山时，这里是安静的，山间的绿意浓稠，空气肃穆而湿润，向警予烈士陵园就在山顶的入口处，再往前走，我们穿过林间路径，看到了龟山电视塔，这座上世纪八十年代建成的庞然巨物，静默地俯瞰着江汉合流的波涛，也见证着对岸灯火的演变。
Continued...
每个人来这里的目的不一样，我们来这里，是来喂小猫的，沿着那条新修的柏油山路，你在跟我讲你学了哪些石头，然后我们就看到叽里呱啦了，诶叽里呱啦并不是算得上可爱的小猫，瘦瘦的，身上毛色没有光亮，那个时候你不敢摸小猫，小猫也怕人，看到我们靠近就往身后躲闪，我们把火腿肠剥开给叽里呱啦吃，叽里很不乖把火腿肠当球踢，呱啦则一下子投入到吃饭的事业中，其实这真的很普通的一天，可是因为这一天是我们第一次喂小猫，它也真的是很重要的一天。

"To travel is to live."- Hans Christian Andersen
Luoyang · Chapter I
洛阳，这座承载着十三朝兴衰的古都，成为了我们当前旅途的最后一站。当我们谈论旅行时，我们总是在感受一座城市的律动，感受自己在大街小巷间情绪的呼吸，可是无论我们谈论什么，我们始终绕不开的是结束，每次旅行都有结束，最后一次当然更让人萦怀，你每次都轻声对我呢喃：“如果能够一直在这里玩，永远不回去该多好呀。”
Luoyang · Chapter II
小皇帝，我也这么想，就算是两个人静静的坐在列车上发呆，都好过跌回原来按部就班的日子里。我们从前一起看瑞克和莫蒂，有一集，恐惧洞在路边的饭店里和莫蒂讨论幸福，它说，幸福本质上是悲哀的，因为所有的幸福都注定会结束，哪怕是最完美的爱情，幸福最好的模样也不过是两个人互相陪伴着，慢慢变老，然后看着彼此死去。
Luoyang · Chapter III
我并不认同这句话，从悲伤的角度出发，结果自然是悲伤的，我却觉得，幸福的意义从来就不在于它是否能永远延续，而在于我们曾真切地拥有过。恐惧洞只看到了生命终点那不可避免的消亡，却不懂得相爱之人并肩走过的每一段路，本身就足以抵抗那种虚无，初上列车的困倦，来到陌生城市的兴奋，出片时的喜悦，面对美丽风景的由衷欣赏，旅途结束的不舍，这些看似微不足道的瞬间，恰恰构成了我们生命中最丰盈的血肉，我们就这样快步走着，就像那天龙门石窟下着小雨，景区门口那条路，长长的。

Spring · Chapter I
老图书馆的青砖黛瓦看透了百年的岁月，而樱顶飘落的花瓣见证了我们的瞬间，古老与新生的交替中，你在我身边的每一天都如春日般明媚；我们曾踏足许多城市的喧嚣，看过不同烟火与风景，可珞珈山、东湖水，从一开始，就更像一种生活，教五草坪和信操总收留学业疲惫，我带你驶在弘毅大道上，我爱看香樟和法桐的叶子层层叠叠，光从叶梢中偷偷溜出来，光是黑夜里的绿意盎然，而绿色是点亮平凡生活的一盏灯。
Spring · Chapter II
我依然记得那天我们在梅操的看台上并肩而坐。912操场附近似乎总能把所有的喧嚣隔绝在外，留下一种剥离了时间的安静。四周没有人，只有一个同学在练习电动车，看着他在夜色里孤单地绕着圈，我的心里突然涌起一种莫名的恻隐，当一个人被巨大且确切的幸福充盈时，他的心是会变得格外柔软的，忘了自己也是世间形形色色的奔波的一员，而沉浸在此刻的圆满中了。
